## 🔧 **المميزات التقنية**

### **تقنيات الخلفية:**
- **Flask**: إطار ويب خفيف وقوي
- **Socket.IO**: اتصال ثنائي الاتجاه في الوقت الفعلي
- **psutil**: مكتبة مراقبة النظام
- **Bootstrap**: إطار واجهة المستخدم المتجاوب

### **تقنيات الواجهة الأمامية:**
- **HTML5/CSS3**: معايير الويب الحديثة
- **JavaScript ES6**: ميزات JavaScript الحديثة
- **Chart.js**: تصور البيانات التفاعلي
- **Bootstrap 5**: مكونات واجهة احترافية

### **ميزات الوقت الفعلي:**
- **اتصال WebSocket**: اتصال مستمر بين العميل والخادم
- **مقاييس مباشرة**: مراقبة النظام في الوقت الفعلي
- **تنبيهات فورية**: إشعارات أمنية فورية
- **تحديث تلقائي**: تحديث البيانات تلقائياً

## 🎯 **كيفية الاستخدام**

### **1. مراقبة النظام:**
1. افتح http://localhost:8080
2. شاهد مقاييس الأداء المباشرة
3. راقب الرسوم البيانية الحية
4. تحقق من حالة الأمان

### **2. تحليل كيان:**
1. انقر على "Analyze Entity"
2. أدخل بيانات الكيان
3. انقر "Analyze"
4. راجع النتائج والتوصيات

### **3. حظر IP:**
1. انقر على "Block IP"
2. أدخل عنوان IP
3. انقر "Block IP"
4. تم حظر العنوان بنجاح

### **4. إنشاء تقرير:**
1. انقر على "Generate Report"
2. انتظر إنشاء التقرير
3. قم بتنزيل التقرير
4. راجع توصيات الأمان

## 🚀 **النشر الإنتاجي**

### **النشر المحلي:**
```bash
cd web_dashboard
python run_web_dashboard.py
```

### **النشر باستخدام Gunicorn:**
```bash
pip install gunicorn eventlet
gunicorn --worker-class eventlet -w 1 --bind 0.0.0.0:8080 app:app
```

### **النشر باستخدام Docker:**
```dockerfile
FROM python:3.9-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 8080
CMD ["python", "run_web_dashboard.py"]
```
