# Deprecated — CLI Dashboards

The terminal dashboards in this folder (`dashboard.py`, `dashboard_simple.py`, `server_control_center*.py`, etc.) are **legacy**.

**Use the web dashboard instead:**

```bash
cd web_dashboard
python run_web_dashboard.py
```

Open: http://127.0.0.1:8080/web_dashboard

These CLI tools remain for offline debugging only and are not the supported operational interface.
