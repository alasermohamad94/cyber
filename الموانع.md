# موانع العمل — ما لا يمكن إكماله تلقائياً

## 1. تدريب نماذج ML إنتاجية (LSTM / XGBoost كامل)
**السبب:** لا توجد في المشروع بيانات موسومة (labels) لحوادث مؤكدة ولا تيليمتري تاريخي كافٍ.  
**ما تم بدلاً منه:** Isolation Forest على أنماط اصطناعية baseline + Shadow Mode + Canary اختياري للثقة فقط.  
**للإكمال:** جمع بيانات حقيقية + تسمية الحوادث + خط أنابيب تدريب منفصل.

## 2. جدار الحماية على مستوى النظام
**السبب:** `netsh` / `iptables` يتطلبان صلاحيات مسؤول (Administrator / root).  
**ما تم:** تسجيل الحظر في SQLite + محاولة قاعدة جدار ناري؛ الحقل `firewall_applied` يوضح النجاح.  
**للإكمال:** تشغيل الخدمة بحساب له صلاحيات الشبكة أو تكامل مع NGFW/API خارجي.

## 3. طوابير موزعة (Celery / RabbitMQ / Redis)
**السبب:** بنية تحتية خارجية غير متوفرة في المشروع الحالي.  
**ما تم:** `workers/background_queue.py` — طابور داخل العملية (in-process).  
**للإكمال:** نشر broker وعامل منفصل عند التوسع الأفقي.

## 4. حذف لوحات CLI بالكامل
**السبب:** قد تُستخدم محلياً للتصحيح؛ الحذف قد يكسر سكربتات قديمة.  
**ما تم:** `dashboard/DEPRECATED.md` + توثيق الويب كواجهة رسمية.  
**للإكمال:** قرار إداري بإزالة المجلد في إصدار major.

## 5. AuthZ خارجي (LDAP / OAuth / MFA)
**السبب:** لا يوجد مزود هوية مهيأ.  
**ما تم:** 3 أدوار محلية (admin / analyst / viewer) عبر متغيرات البيئة.  
**للإكمال:** ربط IdP مؤسسي عند النشر.

## 6. اختبارات ضغط وتزامن Socket.IO شاملة
**السبب:** تتطلب بيئة CI متعددة العمليات وحملاً محاكياً.  
**ما تم:** 12+ اختبار وحدة/تكامل أساسي.  
**للإكمال:** إضافة Locust أو pytest-xdist في pipeline CI.
