## 1. تدريب نماذج ML إنتاجية (LSTM / XGBoost كامل)

**السبب:** لا توجد بيانات موسومة (labels) ولا تيليمتري تاريخي كافٍ.  
**ما تم:** Isolation Forest + Shadow Mode + Canary للثقة فقط.  
**للإكمال:** جمع حوادث موسومة + pipeline تدريب.

---

## 2. جدار الحماية على مستوى النظام

**السبب:** `netsh` / `iptables` يتطلبان Administrator / root.  
**ما تم:** SQLite + `firewall_applied`. على **Render** يُوصى بـ `CDS_FIREWALL_ENABLED=false`.  
**للإكمال:** تشغيل بصلاحيات شبكة أو NGFW خارجي.

---

## 3. طوابير موزعة (Celery / RabbitMQ / Redis)

**السبب:** لا broker في المشروع.  
**ما تم:** `workers/background_queue.py` (داخل العملية).  
**للإكمال:** نشر broker عند التوسع الأفقي.

---

## 4. حذف لوحات CLI بالكامل

**السبب:** سكربتات قديمة قد تعتمد على `dashboard/`.  
**ما تم:** `dashboard/DEPRECATED.md` — الواجهة الرسمية `web_dashboard/`.  
**للإكمال:** قرار إداري + إصدار major.

---

## 5. AuthZ خارجي (LDAP / OAuth / MFA)

**السبب:** لا مزود هوية مهيأ.  
**ما تم:** 3 أدوار محلية (admin / analyst / viewer) مع RBAC كامل في API والواجهة.  
**للإكمال:** ربط IdP مؤسسي.

---

## 6. اختبارات ضغط وتزامن Socket.IO شاملة

**السبب:** يحتاج CI متعدد العمليات وحملاً محاكياً.  
**ما تم:** **22** اختبار وحدة/تكامل (يشمل RBAC و403 وتلاعب الجلسة).  
**للإكمال:** Locust أو pytest-xdist في pipeline.
