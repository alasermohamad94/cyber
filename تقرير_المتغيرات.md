## ملخص تنفيذي

تم تطبيق الأولويات **P0 → P2** من التقرير المرجعي، مع أساس **P3** (Shadow stub)، ثم إصلاحات **تشغيل الويب** و**التصميم (UI)**. للمصفوفة الكاملة للتوافق بين PDF ↔ الكود ↔ هذا الملف، راجع التقرير الشامل.

---

## P0 — الأمان التشغيلي الفوري

| المتغير | قبل | بعد |
|---------|-----|-----|
| مصادقة API | غير موجودة | جلسة Flask + `@api_login_required` على `/api/*` الحساسة |
| AuthZ (أدوار) | — | **منفّذ** — admin / analyst / viewer + صلاحيات API |
| `SECRET_KEY` | ثابت في الكود | `CDS_SECRET_KEY` من البيئة |
| CORS / الربط | `*` و `0.0.0.0` | `CDS_CORS_ORIGINS` و `CDS_BIND_HOST` (افتراضي `127.0.0.1`) |
| حظر IP | تسجيل حدث فقط | `security/firewall.py` + SQLite؛ `firewall_applied` |
| تسجيل الدخول | — | `login.html` + `/login` `/logout` |

**ملفات جديدة:** `security/`, `storage/`, `.env.example`
